1.运行install.bat。
2.运行schedule.bat,自动定时启动与关闭，可通过编辑此文件指定任意复数时间段。
3.对rule.ini进行编辑
username：设置的用户名
password：用户名对应的密码
spot：设置的地点关键词
keyword：设置的美容相关关键词
famous_person：设置的名人关键词
key_order：搜索关键词时定义的初始序号（若想从第一个关键词开始搜索，则设置为0）
fans_order：搜索关键词时定义的初始序号（若想从第一个关键词开始搜索，则设置为0）
spot_order：搜索关键词时定义的初始序号（若想从第一个关键词开始搜索，则设置为0）
model：模式选择 1为对地点关键词进行搜索 2为对美容相关关键词进行搜索 3为对名人关键词进行搜索
4.运行run.bat启动

运行需要将浏览器默认语言设置为英文
